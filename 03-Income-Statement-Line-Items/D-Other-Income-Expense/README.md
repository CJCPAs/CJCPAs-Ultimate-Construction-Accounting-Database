## Overview: Other Income/Expense

### Definition

Income and expenses not related to primary construction operations.

Also called:
- Non-Operating Income/Expense
- Below-the-Line Items
- Other Items

---

### Income Statement Placement

```
Operating Income (Income from Operations)
  + Other Income
  - Other Expense
  = Net Interest Income (Expense)
─────────────────────────────────────────
Net Income Before Income Taxes
  - Income Tax Expense
─────────────────────────────────────────
Net Income
```

---

## 1. Interest Income

### Definition

Income earned from interest-bearing accounts and investments.

---

### Sources in Construction Companies

| Source | Description |
|--------|-------------|
| Bank interest | Savings, money market accounts |
| Investment interest | Short-term investments |
| Notes receivable | Interest on loans to others |
| Customer late fees | Interest on late payments |

---

### Journal Entry

**Accrue Interest Income:**
```
Dr. Interest Receivable                    500
    Cr. Interest Income                          500
```

**Receive Interest:**
```
Dr. Cash                                   500
    Cr. Interest Receivable                      500
```

---

### Balance Sheet / Income Statement

**Income Statement:**
```
OTHER INCOME (EXPENSE):
  Interest Income                          $5,000
```

## 2. Interest Expense

### Definition

Cost of borrowed funds.

---

### Sources in Construction Companies

| Source | Description |
|--------|-------------|
| Line of credit | Working capital borrowing |
| Equipment loans | Equipment financing |
| Mortgages | Real property loans |
| Vehicle loans | Fleet financing |
| Related party | Shareholder loans |

---

### Interest Rate Types

| Type | Description |
|------|-------------|
| **Fixed** | Same rate throughout term |
| **Variable** | Adjusts with prime/LIBOR/SOFR |
| **Prime +** | Prime rate plus spread |

---

### Imputed Interest

**When Related Party Loans Have No/Low Interest:**

IRS may impute interest at AFR (Applicable Federal Rate).

**Example:**
- Shareholder loan: $100,000
- Stated rate: 0%
- AFR: 4%
- Imputed interest: $4,000/year

**Journal Entry:**
```
Dr. Interest Expense                     4,000
    Cr. Due to Shareholder                      4,000
```

---

### Journal Entry

**Accrue Interest Expense:**
```
Dr. Interest Expense                     2,500
    Cr. Accrued Interest                        2,500
```

**Pay Interest:**
```
Dr. Interest Expense                     2,500
    Cr. Cash                                    2,500
```

**Interest on LOC (Combined Principal & Interest):**
```
Dr. Line of Credit                      10,000
Dr. Interest Expense                     1,500
    Cr. Cash                                   11,500
```

---

### Income Statement Presentation

```
OTHER INCOME (EXPENSE):
  Interest Expense                       $(45,000)
```

Or detailed:
```
OTHER EXPENSE - INTEREST:
  Line of Credit                      $(12,000)
  Equipment Loans                      (18,000)
  Mortgage                             (10,000)
  Other Interest                        (5,000)
                                      ──────────
  Total Interest Expense              $(45,000)
```

## 3. Gain/Loss on Sale of Assets

### Definition

Profit or loss from selling fixed assets.

---

### Calculation

```
Sale Proceeds
- Net Book Value (Cost - Accumulated Depreciation)
= Gain (Loss) on Sale
```

---

### Example - Gain

| Item | Amount |
|------|--------|
| Original Cost | $150,000 |
| Accumulated Depreciation | $100,000 |
| Net Book Value | $50,000 |
| Sale Price | $65,000 |
| **Gain** | **$15,000** |

**Journal Entry:**
```
Dr. Cash                                65,000
Dr. Accumulated Depreciation           100,000
    Cr. Equipment                             150,000
    Cr. Gain on Sale of Equipment              15,000
```

---

### Example - Loss

| Item | Amount |
|------|--------|
| Original Cost | $150,000 |
| Accumulated Depreciation | $100,000 |
| Net Book Value | $50,000 |
| Sale Price | $35,000 |
| **Loss** | **$(15,000)** |

**Journal Entry:**
```
Dr. Cash                                35,000
Dr. Accumulated Depreciation           100,000
Dr. Loss on Sale of Equipment           15,000
    Cr. Equipment                             150,000
```

---

### Trade-Ins

**Under GAAP:** Generally recognize gain or loss

**Example - Trade with Boot:**
| Item | Amount |
|------|--------|
| Old Equipment Cost | $100,000 |
| Old Accumulated Depreciation | $80,000 |
| Old Net Book Value | $20,000 |
| Trade-In Allowance | $25,000 |
| New Equipment Price | $150,000 |
| Cash Paid (Boot) | $125,000 |
| **Gain** | **$5,000** |

**Journal Entry:**
```
Dr. Equipment (new)                    150,000
Dr. Accumulated Depreciation            80,000
    Cr. Equipment (old)                       100,000
    Cr. Cash                                  125,000
    Cr. Gain on Sale                            5,000
```

---

### Income Statement Presentation

```
OTHER INCOME (EXPENSE):
  Gain (Loss) on Sale of Equipment         $15,000
```


---

**Equipment or other asset impairments:**

**Journal Entry:**

### Goodwill Impairment

**When goodwill is impaired:**

**Journal Entry:**

## 6. Income Taxes

### Overview

**Entity Type Determines Tax Treatment:**

| Entity | Federal Income Tax |
|--------|-------------------|
| C Corporation | Pays corporate tax |
| S Corporation | Pass-through (shareholders pay) |
| LLC (partnership) | Pass-through (members pay) |
| Partnership | Pass-through (partners pay) |
| Sole Proprietorship | Pass-through (owner pays) |

---

### C Corporation Income Tax

**2024 Rate:** 21% flat rate

**Journal Entry - Provision:**
```
Dr. Income Tax Expense                  84,000
    Cr. Income Taxes Payable                   84,000
```

**Quarterly Estimated Payments:**
```
Dr. Income Taxes Payable                21,000
    Cr. Cash                                   21,000
```

---

### Deferred Income Taxes

**Temporary Differences in Construction:**

| Item | Book | Tax | Deferred Tax |
|------|------|-----|--------------|
| Depreciation | Straight-line | MACRS | Liability |
| Revenue | % of completion | Completed contract | Liability |
| Bad debt | Allowance | Direct write-off | Asset |
| Warranty | Accrued | When paid | Asset |

**Journal Entry - Deferred Tax Liability:**
```
Dr. Income Tax Expense                  25,000
    Cr. Deferred Tax Liability                 25,000
```

---

### S Corporation / Pass-Through Entities

**NO corporate income tax** - income flows to shareholders/members.

**Income Statement:**
```
Net Income Before Income Taxes          $400,000
Income Tax Expense                             0
                                       ──────────
Net Income                             $400,000
```

**Note Disclosure:**
"The Company has elected to be taxed as an S corporation under the Internal Revenue Code. As such, income tax liabilities are the responsibility of the individual shareholders."

---

### State Income Taxes

**Varies by state:**
- Some states: No income tax
- Some states: Tax S corps/LLCs
- Franchise taxes may apply

**Journal Entry:**
```
Dr. State Income Tax Expense             8,000
    Cr. State Income Taxes Payable              8,000
```

---

### Income Statement Presentation

**C Corporation:**
```
Net Income Before Income Taxes          $400,000
Income Tax Expense                       (84,000)
                                       ──────────
Net Income                             $316,000
```

**S Corporation / LLC:**
```
Net Income Before Income Taxes          $400,000
Provision for Income Taxes                     -
                                       ──────────
Net Income                             $400,000
```

## Summary: Other Income/Expense Key Points

| Item | Key Considerations |
|------|-------------------|
| Interest Income | Proper accrual at period end |
| Interest Expense | Include all debt sources |
| Gain/Loss on Sales | Calculate correctly, proper entries |
| Other Income | Proper classification |
| Other Expense | Distinguish from operating |
| Income Taxes | Entity type determines treatment |
