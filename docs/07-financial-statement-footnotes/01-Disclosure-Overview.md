# Financial Statement Footnote Disclosures Overview

## Introduction

This section provides a **comprehensive library of financial statement footnote disclosure templates** tailored for construction companies. These templates are designed as GAAP-compliant starting points that require customization to fit specific client circumstances. Professional judgment is always required when applying these templates.

## Purpose of Financial Statement Disclosures

Financial statement disclosures serve to:

1. **Amplify** - Provide additional detail about amounts presented in the financial statements
2. **Explain** - Clarify accounting policies and methods used
3. **Supplement** - Present information not captured in the primary statements
4. **Alert** - Notify users of risks, uncertainties, and contingencies
5. **Comply** - Meet specific GAAP disclosure requirements

## How to Use These Templates

### Template Notation

Throughout this library, you will see the following notation:

- `[COMPANY NAME]` - Replace with entity name
- `[YEAR]` / `[PRIOR YEAR]` - Replace with applicable years
- `[DATE]` - Replace with specific date
- `[AMOUNT]` - Replace with dollar amounts
- `[DESCRIPTION]` - Replace with specific description
- `[XX]` - Replace with applicable number
- `{OPTIONAL: text}` - Include if applicable to the entity
- `{CHOOSE ONE: option A | option B}` - Select appropriate option
- `{IF APPLICABLE: text}` - Include only if situation exists

### Customization Guidelines

1. **Read the entire template** before customizing
2. **Evaluate applicability** of each paragraph to the specific entity
3. **Remove inapplicable sections** rather than leaving blanks
4. **Add entity-specific details** where templates provide general language
5. **Verify numerical accuracy** of all amounts
6. **Cross-reference** to related disclosures where appropriate
7. **Update terminology** for entity type (corporation vs. LLC vs. partnership)

### Entity Type Considerations

| Entity Type | Equity Section Title | Distribution Term | Tax Note Considerations |
|-------------|---------------------|-------------------|------------------------|
| C Corporation | Stockholders' Equity | Dividends | Full income tax disclosure |
| S Corporation | Stockholders' Equity | Distributions | Pass-through notation |
| LLC (Member-managed) | Members' Equity | Distributions | Pass-through notation |
| LLC (Manager-managed) | Members' Equity | Distributions | Pass-through notation |
| Partnership | Partners' Capital | Distributions | Pass-through notation |
| Sole Proprietorship | Owner's Equity | Draws | No separate tax disclosure |

- [ ] Nature of operations

- [ ] Revenue recognition policy and method
- [ ] Contract assets and liabilities
- [ ] Accounts receivable (including retainage)
- [ ] Property, plant, and equipment

## Materiality Considerations

Disclosures should be evaluated for materiality considering:

1. **Quantitative factors**
   - Percentage of total assets
   - Percentage of total revenue
   - Percentage of net income
   - Dollar magnitude

2. **Qualitative factors**
   - Nature of the item
   - Circumstances giving rise to it
   - Potential impact on users' decisions
   - Regulatory or contractual requirements

## Common Disclosure Pitfalls

| Pitfall | Solution |
|---------|----------|
| Boilerplate language without customization | Tailor each disclosure to specific facts |
| Missing required quantitative data | Include all required tables and amounts |
| Inconsistent terminology | Use consistent terms throughout |
| Stale disclosures from prior years | Update for current period changes |
| Over-disclosure of immaterial items | Focus on material matters |
| Under-disclosure of significant items | Ensure completeness for key areas |
| Missing cross-references | Link related disclosures |
| Incorrect accounting standard references | Verify current ASC references |
