## 1. Cash and Cash Equivalents

### Audit Objectives
- **Existence:** Cash exists at balance sheet date
- **Completeness:** All cash is recorded
- **Rights:** Company has rights to the cash
- **Valuation:** Cash is properly valued

### Risk Assessment
**Inherent Risk:** LOW (easily verified)
**Control Risk:** Varies by client

### Audit Procedures

| Procedure | Assertion | Description |
|-----------|-----------|-------------|
| **Bank Confirmations** | Existence, Completeness | Send standard bank confirmation to all banks |
| **Bank Reconciliations** | Existence, Completeness | Obtain and test year-end reconciliations |
| **Outstanding Checks** | Existence, Cutoff | Test items > X days outstanding |
| **Deposits in Transit** | Existence | Trace to subsequent bank statement |
| **Cutoff Testing** | Cutoff | Test last receipts and disbursements |
| **Restricted Cash** | Rights, Classification | Inquire about restrictions |

### Bank Confirmation - Key Items to Confirm
- Account balances
- Outstanding loans
- Lines of credit (available and used)
- Compensating balance requirements

### Sample Workpaper - Bank Reconciliation

```

## 2. Contract Receivables

### Audit Objectives
- **Existence:** Receivables exist and are owed to company
- **Completeness:** All receivables recorded
- **Valuation:** Net realizable value (allowance adequate)
- **Cutoff:** Recorded in correct period

### Risk Assessment
**Inherent Risk:** MEDIUM
**Control Risk:** Varies - depends on billing controls

### Audit Procedures

| Procedure | Assertion | Description |
|-----------|-----------|-------------|
| **Confirmations** | Existence, Accuracy | Positive confirmations to customers |
| **Alternative Procedures** | Existence | If no reply - subsequent receipts, contracts |
| **Aging Analysis** | Valuation | Review aging, test accuracy |
| **Allowance Evaluation** | Valuation | Evaluate adequacy of allowance |
| **Cutoff Testing** | Cutoff | Test billings around year-end |
| **Subsequent Receipts** | Existence, Valuation | Review post-YE collections |

### Confirmation Approach

**Sample Selection:**
- All balances over $X (materiality-based)
- Random sample of others
- Focus on old balances, unusual items

**Confirmation Types:**
- **Positive:** Customer must respond (preferred)
- **Negative:** Respond only if disagree (weak)

### Alternative Procedures (If No Reply)

1. Review subsequent cash receipts
2. Examine contract and billing documentation
3. Review shipping/delivery documentation
4. Verify to WIP schedule

### Allowance Evaluation

```

## 4. Costs in Excess of Billings (Underbillings)

### Audit Objectives
- **Valuation:** Properly calculated per WIP
- **Existence:** Revenue has been earned
- **Accuracy:** Calculations are correct
- **Cutoff:** Proper period recognition

### Risk Assessment
**Inherent Risk:** HIGH (estimate-driven)

**See [Section 07 - WIP Audit Comprehensive](../../07-WIP-Audit-Comprehensive/) for detailed procedures**

### Key Procedures Summary

| Procedure | Description |
|-----------|-------------|
| **WIP Schedule Tie-out** | Agree underbillings to WIP |
| **Job Cost Testing** | Test costs to supporting documents |
| **ETC Review** | Evaluate estimates to complete |
| **Contract Review** | Verify contract amounts, COs |
| **Recalculation** | Independently recalculate % complete and revenue |
| **Billing Review** | Verify billings per billing system |

### Key Audit Questions
1. Why hasn't this revenue been billed?
2. Is the customer aware of the unbilled amount?
3. Are there disputes affecting billing?
4. Are ETCs realistic?

## 5. Inventory

### Audit Objectives
- **Existence:** Inventory physically exists
- **Valuation:** Properly valued at lower of cost or NRV
- **Completeness:** All inventory recorded
- **Rights:** Company owns the inventory

### Risk Assessment
**Inherent Risk:** LOW-MEDIUM (most contractors have minimal inventory)

### Audit Procedures

| Procedure | Assertion | Description |
|-----------|-----------|-------------|
| **Physical Observation** | Existence | Attend inventory count |
| **Test Counts** | Existence, Accuracy | Perform independent counts |
| **Cutoff Testing** | Cutoff | Test receipts/shipments around YE |
| **Valuation Testing** | Valuation | Test costs to invoices |
| **Slow-Moving Review** | Valuation | Identify obsolete items |
| **Allocation Review** | Completeness | Verify not charged to jobs |

### Construction-Specific Issue: Uninstalled Materials

**Key Question:** Are significant materials purchased but not installed?

**If Yes:**
- Should NOT inflate percentage complete
- Consider proper revenue recognition treatment

- **Accuracy:** Amounts are correct
- **Cutoff:** Recorded in correct period
